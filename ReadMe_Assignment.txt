# InterviewAssignment

# Framework and Language Used

This project is developed using Angular 15 framework and a JSON backend server.
Angular is a popular JavaScript framework for building web applications, and it provides a powerful set of
tools and features for creating responsive and dynamic user interfaces.

The JSON backend server is assumed to be implemented using the JSON Server library, which provides a mock API server using a JSON file.

# Running the Code

To run the code and view the login page with validation features, user table with delete functionality,
and sortable columns, follow these steps:

1. Clone or download the project from the repository.

2. Ensure that you have the required software and dependencies installed:

    Node.js: Visit the official Node.js website (https://nodejs.org) and
    download the latest stable version for your operating system. Follow the installation instructions specific to your OS.

    Angular CLI: Open a terminal or command prompt and run the following command to install Angular CLI globally:

    npm install -g @angular/cli@15

3. Open a terminal or command prompt and navigate to the project directory.

4. Install the project dependencies by running the following command:

    npm install

5. Start the JSON backend server by running the following command:

    npm run json-server

6. In a separate terminal or command prompt, start the Angular application by running the following command:

    ng serve

7. Open your web browser and visit http://localhost:4200 to access the application.

8. On the login page, enter name and password and click the login button to log in.

9. After logging in, you will be redirected to the dashboard page.
   Here, you will see a user table with the ability to delete rows and sortable columns.

Remember to consult the Angular documentation (https://angular.io/docs) for any additional information or troubleshooting
assistance related to Angular-specific features and functionalities.

If you encounter any issues or have further questions, please don't hesitate to reach out for support.
